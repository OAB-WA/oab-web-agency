- **Exact Match [keyword]**: Only shows for exact search
- **Phrase Match "keyword"**: Shows when phrase is included

## üìà OPTIMIZATION STRATEGY: Week-by-Week

### **Week 1-2: Learning Phase**
- **Goal**: Let Google gather data
- **Actions**:
  - Don't make major changes
  - Monitor search terms report daily
  - Add negative keywords for irrelevant searches
  - Check which ads are performing
- **Expected**: 0-2 leads (this is normal!)

### **Week 3-4: Optimization Phase**
- **Goal**: Refine based on data
- **Actions**:
  - Pause low-performing keywords (0 clicks after 100+ impressions)
  - Increase bids on high-performing keywords
  - Test new ad variations
  - Add more negative keywords
  - Adjust ad schedule if you see time patterns
- **Expected**: 1-3 leads

### **Week 5+: Scaling Phase**
- **Goal**: Maximize conversions
- **Actions**:
  - Switch to "Maximize Conversions" bidding (if 15+ conversions)
  - Double down on winning keywords
  - Expand similar keywords
  - Test landing page variations
  - Consider increasing budget if profitable
- **Expected**: 2-5 leads/week
